# v0.1.1

Release packaging improvement.

- Release assets are ready-to-run packages.
- macOS/Linux packages include `run.sh`.
- Windows package includes `run.ps1` and `run.cmd`.
- Each package includes `RUNNING.md` quick-start notes.
- Users do not need Rust or a compilation step when downloading release assets.

# v0.1.0

Initial public release.

- Static GitHub Pages web app.
- Rust CLI for local YAML generation.
- Accademia template presets for Mobile, Desktop, Universal, and BlackList variants.
- Missing-node strategy groups can be hidden with `hidden: true`.
- Routing fields are preserved: `rules`, `rule-providers`, and `dns.nameserver-policy`.
- Credit, privacy, disclaimer, and legal-use documents included.
