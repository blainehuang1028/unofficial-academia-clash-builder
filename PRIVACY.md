# Privacy Policy

## Static Web App

The GitHub Pages web app is static. It has no project-operated backend server and does not store uploaded files, pasted YAML, generated YAML, node names, server addresses, subscription URLs, or usage history.

YAML transformation happens in your browser memory.

## URL Loading

If you enter a template or node URL, your browser sends a request directly to that URL. The remote host may receive normal request metadata such as your IP address, user agent, time of request, and requested URL.

The GitHub Pages version does not proxy remote URLs. If a custom URL blocks browser CORS access, use file upload or paste the YAML manually.

## Local CLI

The Rust CLI runs locally. It reads local files or URLs you provide and writes output to the path you choose. It does not send telemetry.

## GitHub Pages Hosting

The live site is hosted by GitHub Pages. GitHub may collect standard platform logs according to GitHub's own policies.
